# SkillVault-web

🔍 A full-stack resume analyzer web application that:

- Matches resumes to job roles
- Shows found + missing skills
- Suggests learning resources
- Includes login, signup, password reset via email

🚀 Built with:
- Python + Flask
- SQLite + JSON
- Gmail SMTP
- HTML/CSS
